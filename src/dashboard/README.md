# Dashboard Application (`src/dashboard/`)

This directory houses the presentation layer of the application: a Flask web server and a dynamic frontend built with Chart.js.

## `app.py`
The backend Flask server. It serves two primary purposes:
1.  **Serve Static Data:** Reads the JSON files generated by PySpark in the `output/` folder and exposes them as RESTful API endpoints for the frontend.
2.  **Proxy GitHub API:** Handles requests to the live GitHub REST API to avoid exposing client-side credentials and to manage CORS/rate limits.

**Key Snippet (Serving Spark Data):**
```python
@app.route('/api/language-popularity')
def language_popularity():
    # Reads the processed data and serves it directly to the dashboard
    with open('../output/1_language_popularity.json', 'r') as f:
        return jsonify(json.load(f))
```

## `templates/index.html`
The main structure of the dashboard. It uses a modern, editorial aesthetic. It organizes the 8 distinct analytical phases into clean, full-width sections.

## `static/style.css`
Contains the custom "hand-crafted" styling. It utilizes CSS variables for a cohesive palette (DM Sans typography, deep backgrounds, and ambient glows), completely avoiding generic CSS frameworks.

## `static/dashboard.js`
The client-side logic. It fetches data from our Flask APIs and renders complex visualizations using `Chart.js`.

**Key Snippet (Dynamic Canvas Replacement):**
To prevent chart overlapping and rendering bugs when users switch years, we completely destroy and recreate the canvas elements:
```javascript
function createChart(id, config) {
    if (chartInstances[id]) { chartInstances[id].destroy(); }
    
    // Replace the canvas element to guarantee a clean rendering context
    const oldCanvas = document.getElementById(id);
    if (oldCanvas) {
        const parent = oldCanvas.parentNode;
        const newCanvas = document.createElement('canvas');
        newCanvas.id = id;
        parent.replaceChild(newCanvas, oldCanvas);
        chartInstances[id] = new Chart(newCanvas, config);
    }
}
```
